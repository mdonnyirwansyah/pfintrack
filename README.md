# PFinTrack

**Personal Finance Tracker — Mobile-First, Privacy-First**

Catat dompet, transaksi, utang-piutang, dan laporan keuanganmu — tanpa akun, tanpa server, semua tersimpan di browser kamu.

## Fitur Utama

| Modul | Fitur |
|---|---|
| **Dompet** | Multi-wallet (bank, e-wallet, investasi, tabungan, aset digital) · Riwayat perubahan saldo |
| **Transaksi** | Pemasukan, pengeluaran, dan transfer antar dompet · Export ke Excel · Navigasi per tanggal |
| **Pinjaman** | Catat memberi & menerima pinjaman · Status lunas/belum lunas · Ringkasan per orang |
| **Laporan** | Realtime · Bulanan · Custom range · Donut chart per kategori · Insight pengeluaran |
| **PWA** | Install di HP, bisa dipakai offline |

---

## Tech Stack

```
Next.js 16 (App Router + Turbopack)
React 19
TypeScript 5 — strict mode
Tailwind CSS v4
Zustand — state management
next-intl — bilingual (ID / EN)
Serwist — PWA & service worker
Recharts — visualisasi laporan
Playwright — end-to-end testing
```

---

## Menjalankan Lokal

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build production
npm run build

# Lint
npm run lint
```

Buka [http://localhost:3000](http://localhost:3000) di browser.
